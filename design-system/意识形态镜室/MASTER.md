# 意识形态镜室设计系统

## Design read

面向高反思意愿中文用户的自我探索产品。它应当像一台安静、可争辩的思想仪器，
而不是心理测评、聊天机器人、哲学百科或占卜产品。

## Design dials

- `DESIGN_VARIANCE: 7`
- `MOTION_INTENSITY: 5`
- `VISUAL_DENSITY: 4`

首页允许不对称与空间感，产品流程优先可读性和注意力。动效只表达显影、分裂、
聚拢、开放和修正，不为装饰而运动。

## Visual language

- 基础色：冷纸白 `#f2f3f1`、烟银 `#d7dad8`、石墨 `#232625`。
- 强调色：暗朱红 `#9f3f32`，全站只使用这一种强调色。
- 暗色模式：墨灰 `#151817`、雾白 `#e6e8e5`、暗朱红保持辨识度。
- 标题使用中文宋体系统栈，表达“阅读与推敲”；正文使用现代中文黑体系统栈。
- 坐标和短标签使用等宽字体，但不让技术语气压过人的表达。
- 外层容器保持近乎直角，按钮与输入使用 10px 圆角，规则必须一致。
- 纹理只由固定伪元素提供轻微纸张颗粒，不能放在滚动容器上。

## Four mirrors and one window

- 场域决定空间规则和背景网格。
- 本体决定视觉重量与锚点。
- 现象决定文字如何显现、重叠或断裂。
- 目的决定元素运动与指向。
- 窗检查权力差、现实边界和不可内化的外部条件。

四种力量不用四种颜色表达：秩序是网格，冲突是裂缝，中心是引力，开放是边界消散。

## Interaction rules

- 一屏只处理一个主要问题。
- 每个系统判断必须绑定用户原话。
- 每条判断都允许选择：像我、不像、只在这里、我有反例。
- 未确认内容保持模糊，不伪造完整坐标。
- 坐标编号和主义名称默认折叠。
- 任何高风险表达优先显示现实支持信息，不继续做哲学归因。
- 所有按钮至少 44px，焦点可见，错误使用 `role="alert"`。
- 动画只改变 `transform` 与 `opacity`，并尊重 `prefers-reduced-motion`。

## Avoid

- 紫蓝 AI 渐变、霓虹外发光、过量玻璃拟态。
- AI 头像、左右聊天气泡、三点输入动画。
- 雷达图、分数条、人格排行、巨大四位编号。
- 旋转三维魔方和把 256 个坐标铺在首页。
- 默认 shadcn 卡片墙、三个等宽功能卡、装饰性状态圆点。
- 神秘符号、塔罗式文案、虚假的全知感。
- 页面可见文案中的长破折号。

## Responsive and themes

- 1440px：不对称 12 列布局，镜面与问题并列。
- 1024px：缩减留白，保留两列。
- 768px 以下：严格单列，镜面改为横向可聚焦序列。
- 375px：正文不小于 16px，输入区不被软键盘遮挡。
- 使用 CSS 语义变量支持系统明暗主题，信息层级在两种主题中保持一致。
