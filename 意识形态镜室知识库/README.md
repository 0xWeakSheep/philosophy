## 使用原则

1. 坐标用于定位，不用于定型。
2. 数字不是道德评分；`4` 不天然高于 `1`，它只是表示更强的开放性、失败经验或不可规定性。
3. 一个用户可以在不同议题上落入不同坐标，例如关系困境是 `2-3-2-1`，职业选择可能是 `1-1-3-2`。
4. AI 对话应当围绕追问展开，而不是直接宣布“你就是某种主义”。
