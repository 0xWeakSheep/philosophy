# 资料来源与使用边界

本库是为 [[意识形态镜室知识库]] 做的产品化知识库，不是未明子课程笔记的逐字转写。

## 本地来源

- [[产品定位]]：当前项目中的产品定位文档，是“自我认知化转译”的主要依据。

## 联网参考

- [主义主义魔方](https://ismismtag.com/)：用于理解四维编码的可视化组织方式。
- [spaceater/ismismcube-frontend](https://github.com/spaceater/ismismcube-frontend)：其 README 说明该仓库包含“主义主义简要总结的 json 文件”；本库用其 JSON 作为坐标名、英文名、轴标签和相关视频链接的数据底座。
- [主义主义笔记](https://zhuanlan.zhihu.com/p/670149536)：用于校准四个维度和四种基本力量的基础定义。
- [主义主义转文字合集](https://zhuanlan.zhihu.com/p/624341227)：用于核对坐标名和转文字资料索引。
- [未明子哲学观点记录（一）](https://arbel000.github.io/2021/04/11/wmz-philosophy1/)：用于补充总纲中场域、本体、现象/认识、目的的解释。
- [ismism-website/ismism-wiki](https://github.com/ismism-website/ismism-wiki)：用于交叉参照部分谱系目录和代表人物。

## 使用边界

1. 页面中的“产品化读法”“追问入口”“可能能力与盲区”是基于原框架做的再解释，不应当被当成未明子原意。
2. 页面中的“对应主义”来自公开资料中的坐标命名，只作为思想史参照，不构成心理诊断或政治判定。
3. 当某些坐标页的原资料较少时，本库仍会根据四维组合生成基础解释；这类页面应在后续阅读原视频或笔记后继续修订。
4. 若用于真实产品，对外展示时应避免直接给用户贴“主义”标签，而应把坐标转化为问题、镜像和可选择的下一步。
