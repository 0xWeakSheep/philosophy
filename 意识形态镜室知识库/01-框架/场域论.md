# 场域论

> 世界或关系场的默认背景

它不是某个具体对象，而是让对象、规则和意义能够出现的“场”。在产品里，它对应用户对世界、关系、社会、命运或系统的默认想象。

返回：[[四维总览]] / [[意识形态镜室知识库]]

## 四种取值

| 取值          | 含义                           | 追问入口                           |
| ----------- | ---------------------------- | ------------------------------ |
| [[1 秩序]]    | 世界被预设为一个稳定、同质、可通约的整体。        | 你说“事情本来如此”时，那个“本来”是谁给你的？       |
| [[2 冲突]]    | 世界被预设为由断裂、对立或双重结构支撑。         | 你看到的冲突是现实本身的结构，还是某个旧秩序崩开后的投影？  |
| [[3 中心]]    | 世界被预设为围绕某个主体、共同体或中介中心组织。     | 如果世界必须围绕某个中心运转，这个中心为什么有资格成为中心？ |
| [[4 虚无-开放]] | 世界被预设为没有最终封闭的背景，只能在开放和失败中显现。 | 当背景不再保证意义时，你还能凭什么继续判断？         |
